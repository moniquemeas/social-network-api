  # Installation
  - Clone the repository from Github.
  - Install Node.js
  - Install MongoDB
  - npm install express, moment and mongoose


  # Technology:

  * JavaScript
  * Node.js
  * MongoDB
  * express.js 
  * moment.js 
  * mongoose
